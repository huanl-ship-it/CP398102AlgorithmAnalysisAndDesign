# CP398102AlgorithmAnalysisAndDesign
Term Project
Student Management System

## How to run:
### option1
1. Install Python 3.x
2. Open terminal or command prompt
3. Run:

   python studentsmanagementsystem.py
   
### option2
Double click StudentManager.exe to run.


## Features:

* Add / Update / Delete student
* Search student
* Data saved automatically

Note:
Data is stored in user's home directory under "StudentManager".
